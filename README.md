# Luiz Ramos 👋

**Designer Estratégico & Desenvolvedor Front-end**

Sou focado em criar interfaces que unem um visual bem pensado com código limpo e performático. Gosto de resolver o problema de design no Figma e dar vida a ele no front-end, priorizando JavaScript puro (Vanilla) e animações fluidas, sem depender de frameworks pesados quando não são necessários.

---

### 💻 Stack

- **Design:** Figma, Adobe Photoshop, Adobe Illustrator
- **Front-end:** HTML5, CSS3, JavaScript (Vanilla), GSAP
- **CMS & Automação:** WordPress, Elementor, Notion, Make (Webhooks)

---

### 🚀 O que eu construo

- **Portfólio Customizado:** Site desenvolvido do zero com JS puro e GSAP. Arquitetura baseada em componentes injetados dinamicamente via arrays (simulando um CMS) e captação de leads integrada via API ao Notion.
- **Landing Pages de Alta Conversão:** Desenvolvimento e refatoração de LPs pesadas para código otimizado, focando em performance extrema no Lighthouse e responsividade (ex: Ouro Car, Escudo Mais).
