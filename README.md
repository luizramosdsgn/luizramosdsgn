# Olá, eu sou o Luiz Ramos! 👋

### Designer Estratégico & Desenvolvedor Front-end

Sou um profissional apaixonado por unir o design visual de alto nível com código limpo e performático. Meu foco é criar experiências digitais completas — da concepção da interface (UI/UX) no Figma até a codificação e animação em código puro, garantindo sites leves, responsivos e focados em conversão.

---

### 🚀 O que eu faço?
- **Web Design & UI/UX:** Criação de interfaces modernas, Design Systems consistentes e jornadas de usuário otimizadas.
- **Desenvolvimento Front-end (Vanilla):** Transformo designs complexos em código pixel-perfect utilizando HTML5 semântico, CSS3 (Grid/Flexbox moderno e variáveis/tokens) e JavaScript puro.
- **Animações Premium:** Especialista em dar vida a interfaces estáticas utilizando **GSAP** (GreenSock) para microinterações, scroll suave e transições de página (estilo *Awwwards*).
- **Integrações Inteligentes:** Automação de fluxos de trabalho conectando formulários web (via Webhooks/Make.com) a bases de dados como o **Notion**.
- **Plataformas CMS:** Experiência no desenvolvimento de Landing Pages de alta conversão utilizando WordPress e Elementor.
